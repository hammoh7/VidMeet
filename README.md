# VidMeet
## Development in Progress

### Getting Started

1. Clone the project:
   ```
   git clone https://github.com/hammoh7/VidMeet.git
   ```
2. Run:
   ```
   npm install
   ```
3. Then, run the development server:
   ```
   npm run dev
   ```
4. Open http://localhost:3000 with your browser to see the result
